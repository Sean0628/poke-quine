# poke-quine

`poke-quine` is a cyclic quine of the original Pokemon starters, created to celebrate Pokemon's 30th anniversary.

The quine cycles through:

- Bulbasaur
- Charmander
- Squirtle

After 3 steps, it returns to the original source.

## AA Sources

- Pokemon reference images were taken from [ポケモンずかん](https://zukan.pokemon.co.jp/)
- ASCII art was initially generated with [TheZoraiz/ascii-image-converter](https://github.com/TheZoraiz/ascii-image-converter)
- The final ASCII art was then manually modified to make the cyclic quine fit and work correctly
